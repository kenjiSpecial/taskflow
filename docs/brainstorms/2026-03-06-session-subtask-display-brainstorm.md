# セッション画面にサブタスクを表示する

**日付**: 2026-03-06
**ステータス**: 承認済み

## 何を作るか

セッション画面（SessionDetailPanel / SessionInlineDetail）で、リンクされた親タスクの下にサブタスクを展開表示する。現状はセッションにリンクされた親タスクのみ表示されており、サブタスクの進捗が見えない問題を解決する。

## なぜこのアプローチか

- **API変更不要**: todo-storeの既存`childrenMap`を活用し、フロントエンドのみの変更で実現
- **既存パターンの踏襲**: TasksSectionの展開/折り畳みUIと同じパターンを再利用
- **シンプルさ優先**: サブタスクは読み取り専用（完了操作はタスク画面で行う）

## 主要な決定事項

| 決定 | 選択 | 理由 |
|------|------|------|
| 表示方法 | 親タスク下に展開/折り畳み | TasksSectionと一貫したUX |
| 進捗表示 | 親タスク横に `[2/3]` バッジ | サブタスク折り畳み時も進捗が見える |
| サブタスク操作 | 表示のみ（読み取り専用） | セッション画面の責務を限定 |
| データ取得 | todo-storeのchildrenMap参照 | API変更不要、既存データ活用 |

## スコープ

### 含む
- セッション画面のリンクタスク一覧にサブタスク展開表示を追加
- 親タスクに進捗バッジ表示
- 展開/折り畳みトグル

### 含まない
- サブタスクの完了/未完了操作
- セッションへのサブタスク直接リンク
- API側の変更
